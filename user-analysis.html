Upload Tips

GeoTIFF preferred. Georeferenced images allow the system to auto-detect the region and apply the correct dataset (fMoW or MLRSNet).

Resolution matters. For best results, use imagery at 10m resolution or finer. The model was trained on 0.3–2m aerial data.

Comparing periods. Upload an image of a previously analyzed region to automatically trigger change detection.

Batch upload. Select multiple files at once to queue several regions for sequential analysis.
